# Java Template

Substitute "javatemplate" with your application name, and "com.bavelsoft" with your company path.

You might want to rename the hellofeature directory and its contents.

Entirely replace this file.

* Dagger
* Errorprone
* Nullaway
* Autovalue
* Mapstruct
* Checkstyle
* Log4j2
* Mx4j
